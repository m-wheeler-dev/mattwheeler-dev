"Become a Dasher" Page Clone

I challenged myself to replicate a corporation's web page just by looking at it, and only using HTML, CSS, and JavaScript. I highly recommend this type of challenge for anyone looking to practice or deepen their front-end development knowledge.

Live Demo GitHub Repo

Space Invaders

The goal for this project was to build a web browser version of Space Invaders, and do so with only HTML, CSS, and JavaScript. The HTML canvas element is used to build the game area while giving JavaScript the ability to "draw" objects such as enemies, projectiles, and the player's ship.

Live Demo GitHub Repo

Hover Colors!

Another excellent JavaScript project. JS is used to spawn (create) the boxes, randomize the color when the user hovers over a box, and to fade the color out when the cursor leaves. While HTML and CSS are present in this project, it is almost entirely JavaScript.

Live Demo GitHub Repo

Memory Game

The classic memory game is an excellent way to practice developer skills, this one is built with HTML, CSS, and JavaScript. JS handles the timing, match-checking, score-keeping, and of course it announces your victory after all the matches are found.
